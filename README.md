# 数学建模国赛 + 美赛存档

本仓库包含历年数学建模国赛和美赛的题解，并将不断更新。主要使用的编程语言为 Python，MATLAB 在针对 A 题等涉及物理类建模时使用。Python 的版本建议在 3.10 以上，MATLAB 的版本建议在 R2020a 以上。

> [!IMPORTANT]
> 仓库中每个题目内的 PDF 文件为历年参赛时提交的论文，内容和数据可能包含错误。每个题目内的 README.md 文件为总结多篇优秀论文后的题解，并将不断更新。若需要参考论文写作格式，请以 PDF 文件为准。若需要参考题解，请以 README.md 文件为准。仓库内所有图表、Excel 数据文件均为最新版本。

## 目录结构

以下标注中，一等奖/二等奖/三等奖为实际参加并获奖，优秀奖为实际参加但未获奖，无标注为未参加，但已有题解。

- 2022 南京信息工程大学数模校赛
  - B 题：蘑菇有毒无毒的分类与识别（三等奖）
- 2022 数模国赛
  - C 题：古代玻璃制品的成分分析与鉴别（优秀奖）
- 2023 江苏省五一数模
  - B 题：快递需求分析问题（二等奖）
- 2024 数模国赛
  - C 题：农作物的种植策略
